# Puppin
An upcoming modern local web server solution stack (similar in spirit to XAMPP) for working with PostgreSQL, PHP, and nginx locally, cross-platform

## About the Name
- P is for **PostgreSQL-powered** (database)
- u is for **universal** (cross-platform support)
- p is for **PHP** (language server)
- p is for **platform**
- i is for **in**
- n is for **nginx** (HTTP server)

Altogether, that's "PostgreSQL-powered universal PHP platform in nginx" 🐶

## MVP Target Features (for v0.1.0)
- core tech stack (PostgreSQL, PHP, nginx)
- Puppin home page on HTML document root
- basic control panel with service management, logging, and file exploration, written in C# using the [Avalonia®](https://avaloniaui.net/avalonia) framework
- basic custom webpage-based administration tool for PostgreSQL (akin to tools such as Adminer and phpMyAdmin)
- Windows 10/11 support
- limited initial Linux (Debian-based) support
