# Puppin
An upcoming modern local web server solution stack (similar in spirit to XAMPP) for working with PostgreSQL, PHP, and nginx locally, cross-platform

## About the Name
- P is for **PostgreSQL-powered** (database)
- u is for **universal** (cross-platform support)
- p is for **PHP** (language server)
- p is for **platform**
- i is for **in**
- n is for **nginx** (HTTP server)

Altogether, that's "PostgreSQL-powered universal PHP platform in nginx" 🐶
